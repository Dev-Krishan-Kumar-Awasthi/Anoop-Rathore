# Anoop Rathore - Portfolio Website

Welcome to the official portfolio website of **Anoop Rathore**, an IIT Guwahati Alumnus and esteemed CSE Lecturer at Government Polytechnic College, Sidhi. 

This website was designed and developed by **Krishan Kumar Awasthi** to showcase the professional achievements, academic background, and technical expertise of Anoop Rathore.

## 🌐 Live Website
**Visit the live site here:** [kkawasthi.vercel.app](https://kkawasthi.vercel.app) *(Also accessible via anoop-rathore.vercel.app)*

## 👨‍🏫 About Anoop Rathore
**Anoop Rathore** is a dedicated Computer Science educator with an M.Tech from IIT Guwahati (2015). He specializes in:
- Image & Video Processing
- Invisible Watermarking
- Computer Networks & Data Structures

He provides high-quality, free engineering notes for Diploma and Degree students, serving as a guiding light for students in Madhya Pradesh and beyond.
